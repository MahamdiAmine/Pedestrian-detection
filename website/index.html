About Our App

Pedestrians detection is an Open source application that detects Pedestrians in streets ,it detects people in general.the app realized by a group of two developers in Algeria (Higher National School of Computer Science - ESI ex INI, Algers).

Fully Responsive

The App is fully responsive ,you can use it on images , Live Wam ,Or videos

Stylish Design

Our app respects the UI /UX rules and best practices. with consistent communication, user friendly and User-focused design ...

Clean Code

Our app respect the clean code norms: Naming Conventions,DRY,Tests,Comments,indents deep understanding of the code ... it's totally Free of Charge and you can review the code on github.

Easy To Customize

The UI let you customise , and choose your preferences.

How The App Works?

Collect data

Collect data from various sources ,the data pictures must be clear .

Train the DPNN

train deep learning-based object detection,it uses the Darknet framework ,witch is an open source neural network written in C and CUDA.

Choose how to use it

you can detect object either on videos or WebCam live stream.

Enjoy

All the peoples in the video will be detects , the precision depends on your configurations and of course of the quality of you dataset.
